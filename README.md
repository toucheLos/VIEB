### VIEB

VIEB, Video Interpreter Excluding Bias, is a neuroscience behavioral analysis machine learning tool. 

## Function

# Video Parser
VIEB is inputting a video of a mouse in a top down fashion. Using DeepLabCut, this video is parsed and the movement of the mouse is recorded and output in an 8D matrix (one vector for each point recorded on the mouse).

DeepLabCut: Convert raw video -> standardized pose tensor
